# Repartir

## Ambiente

 * JDK 17
 * Docker
 * Chrome

## Ejecutar pruebas

Para ejecutar todas las pruebas

```
./gradlew check --info
```

```
./gradlew fastAcceptanceTest
```

Regla -> Para que el total sea positivo.

## Para levantar en el ambiente local

```
./gradlew bootRun
```
